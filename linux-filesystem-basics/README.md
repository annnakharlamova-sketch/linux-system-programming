# Linux Filesystem Basics

This repository demonstrates practical work with the Linux filesystem using standard GNU/Linux utilities.

The project covers filesystem hierarchy analysis, file and directory operations, links, and system data extraction.

---

## Topics Covered

- Linux filesystem hierarchy (`/bin`, `/etc`, `/proc`, `/var`, `/usr`, `/lib`)
- File and directory operations
- Hard and symbolic links
- System log analysis
- Extracting system information from `/proc`

---

## Key Commands

`pwd`, `cd`, `ls`, `file`, `man`  
`mkdir`, `touch`, `echo`, `tree`  
`ln`, `chmod`, `du`, `df`, `grep`, `find`, `comm`

---

## Environment

- OS: GNU/Linux
- Shell: Bash
