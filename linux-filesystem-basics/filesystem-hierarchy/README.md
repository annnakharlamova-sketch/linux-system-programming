# Файловая система Linux и её структура

Основные системные каталоги Linux:
- `/bin` – исполняемые файлы команд для всех пользователей.
- `/sbin` – системные исполняемые файлы, используемые администратором.
- `/etc` – конфигурационные файлы системы.
- `/dev` – устройства системы.
- `/proc` – виртуальная файловая система с информацией о процессах.
- `/home` – домашние каталоги пользователей.
- `/var` – переменные данные, логи, базы данных.
- `/usr` – вторичные программы и утилиты.
- `/tmp` – временные файлы.
- `/lib` – системные библиотеки.

Дополнительно изучено:
- Получение справки по иерархии: `man hier`.
- PID текущего shell через `echo $$` и просмотр содержимого `/proc/[PID]`.
