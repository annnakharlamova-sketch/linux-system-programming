# Работа с файлами и каталогами

В рамках лабораторной работы изучались:
- Навигация по файловой системе (`cd`, `pwd`).
- Создание каталогов и файлов (`mkdir`, `touch`, `echo`).
- Просмотр структуры каталогов (`tree`).
- Создание ссылок (`ln`, `ln -s`).
- Проверка содержимого файлов.

Все операции выполнялись в домашнем каталоге студента с созданием собственной структуры каталогов.
